# 变更提案：工作流操作/编辑页 + 后端接口

## 背景
需要交付一个无需登录的“工作流操作/编辑页”，UI 使用 shadcn/ui 风格，中文文案。后端先用 OpenSpec 作为规格文档，确认后再用 FastAPI 实现。

## 目标
- 交付单页 UI（顶部标题/状态/操作按钮、流程画布、右侧参数与节点配置、底部日志/历史）。
- 产出 OpenSpec 规格文档作为后端接口契约。
- 规格确认后再落地 FastAPI 实现。

## 范围
- 本次仅包含“工作流操作/编辑页”的 UI 与接口。
- 仅最小可用接口（列表/详情/日志/运行/保存）。

## 非目标
- 登录/权限体系。
- 复杂运行引擎与执行器实现。
- 跨页面导航与多页面应用结构。
